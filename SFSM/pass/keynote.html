    초기화 주의
    폼을 재생성하면 현재 화면에서 입력 중이던 내용이 모두 사라지고 빈 폼으로 변경됩니다.
    • 새 문서: Ctrl + Alt + N
    • 저장하기: Ctrl + S
    • 문서 검색: Ctrl + E
    • 몰입 모드(UI 숨기기): Ctrl + H
    • 카드 모드(1문항씩 보기): Ctrl + M (좌/우 방향키로 이동)
    • 화면 스크롤: 화살표 위/아래 (입력 중이 아닐 때)
    • 글자 확대/축소: Ctrl + ➕ / ➖
    • 두음 강조(남색+흰글씨): Ctrl + Alt + M
    • 이미지 정렬 및 크기: 이미지를 클릭하면 사이즈 조절 및 정렬 툴바가 표시됩니다.
    • 이미지 크게 보기: 에디터 내 이미지 더블클릭
    • 에디터 서식: 취소선(Ctrl+0) | 빨강(Ctrl+Alt+R) | 파랑(Ctrl+Alt+B)
    • 컬러 형광펜: 주황(Ctrl+1) | 파랑(Ctrl+2) | 노랑(Ctrl+3) | 녹색(Ctrl+4) | 분홍(Ctrl+5) | 갈색(Ctrl+6) | 회색(Ctrl+7)
    • 이미지 강조(테두리): 이미지 클릭 후 정렬 툴바 우측 형광펜(적용) / 지우개(원복) 버튼 클릭
    • 표(Table) 지우기: 표의 맨 앞이나 뒤 클릭 후 백스페이스(지우기)
    • 서식 지우기: Ctrl + Shift + X
